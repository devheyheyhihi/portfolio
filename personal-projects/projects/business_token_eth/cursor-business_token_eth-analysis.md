# Cursor Business Token ETH 프로젝트 분석

## 📋 프로젝트 개요

**BusinessCoin**은 이더리움 블록체인 기반의 ERC-20 표준을 준수하는 커스텀 토큰 프로젝트입니다. Hardhat 개발 환경을 사용하여 구현되었으며, OpenZeppelin 라이브러리를 활용한 안전하고 표준화된 토큰 컨트랙트입니다.

## 🏗️ 프로젝트 구조 분석

### 📁 디렉토리 구조
```
business_token_eth/
├── contracts/           # 스마트 컨트랙트 소스 코드
├── scripts/            # 배포 및 상호작용 스크립트
├── test/              # 테스트 파일
├── artifacts/         # 컴파일된 컨트랙트 아티팩트
├── cache/            # Hardhat 캐시

├── hardhat.config.js # Hardhat 설정

```

- **Ethers.js**: 하드햇 툴박스 포함
- **Chai**: 테스트 프레임워크

## 💎 스마트 컨트랙트 분석

### BusinessCoin.sol 핵심 기능

#### 1. 토큰 기본 정보
- **이름**: BusinessCoin
- **심볼**: BSC
- **최대 공급량**: 100,000,000 BSC (1억 개)
- **초기 발행량**: 30,000,000 BSC (최대 공급량의 30%)

#### 2. 상속된 기능
```solidity
contract BusinessCoin is ERC20, ERC20Burnable, Ownable
```
- **ERC20**: 표준 ERC-20 토큰 기능
- **ERC20Burnable**: 토큰 소각 기능
- **Ownable**: 소유자 권한 관리

#### 3. 주요 함수
- `mint(address to, uint256 amount)`: 소유자 전용 토큰 발행
- `burn(uint256 amount)`: 토큰 소각 (상속)
- `transfer()`: 토큰 전송 (상속)
- `balanceOf()`: 잔액 조회 (상속)

#### 4. 보안 기능
- 최대 공급량 제한 검증
- 소유자 권한 검증 (`onlyOwner`)
