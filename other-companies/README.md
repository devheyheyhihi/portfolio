# 기타 회사 프로젝트

이 디렉토리는 Innerview IT 외 다른 회사에서 작업한 프로젝트들을 정리하는 공간입니다.

## 📁 구조

각 회사별로 폴더를 생성하여 프로젝트를 정리합니다.

```
other-companies/
├── README.md              # 이 파일
├── company-a/             # A 회사 프로젝트
│   ├── README.md
│   └── projects/
├── company-b/             # B 회사 프로젝트
│   ├── README.md
│   └── projects/
└── ...
```

## 🏢 회사 목록

### 1. 현대엔지니어링
**역할**: 시스템 운영 관리자 / 현장 관리자  
**프로젝트**: 현지니 앱 - 건물 관리 시스템

**주요 업무**:
- 100,000건 이상의 대규모 데이터베이스 관리 및 운영
- 현대 계열사 5개사 건물 관리 서비스 제공
- 현장 출장을 통한 건물 구조 파악 및 관리
- 고객사, 하청업체, 내부 팀 간 커뮤니케이션 및 조율
- 간단한 프론트엔드 수정 및 외주 업체 협업 관리

**주요 성과**:
- 법적 리스크 방지를 위한 중개 플랫폼 안정적 운영
- 현대 트랜시스, 모터스 스튜디오, 모비스, 제철, 대한항공 등 관리
- 체계적인 요청사항 처리 시스템을 통한 업무 효율성 향상

📄 [상세 문서 보기](./hyundai-engineering/hyunjini-app-operation.md)

### 2. KTL (한국산업기술시험원)
**역할**: 물품 관리 담당자  
**기관**: 국내 유일 공공 종합시험인증기관

**주요 업무**:
- 엑셀 기반 물품 재고 및 구매 관리 시스템 운영
- VBA 매크로를 활용한 업무 자동화
- 피벗 테이블 및 고급 함수를 활용한 데이터 분석
- 월간/분기별 재고 및 구매 보고서 작성
- 부서 간 협업 및 공급업체 관리

**핵심 역량**:
- 엑셀 고급 함수 및 VBA 매크로 전문성
- 대용량 데이터 정확성 및 체계적 관리
- 업무 프로세스 자동화 및 효율화
- 데이터 시각화 및 보고서 작성

📄 [상세 문서 보기](./ktl/ktl-inventory-management.md)

### 3. NPS 외주업체 (국민연금공단 프로젝트)
**역할**: 빅데이터 분석가 / 데이터 엔지니어  
**협력기관**: 전주시청

**주요 업무**:
- Python 및 Apache Spark를 활용한 대용량 데이터 처리
- 농산물 데이터 분석 (생산, 유통, 소비 패턴)
- 유산균 데이터 분석 (발효 식품, 생산 공정)
- 전주시청과 협력하여 정책 수립 지원
- 데이터 파이프라인 구축 및 자동화

**기술 스택**:
- Python, PySpark, Pandas, NumPy, Scikit-learn
- Apache Spark, Hadoop HDFS
- SQL, Jupyter Notebook
- 통계 분석, 머신러닝, 데이터 시각화

**주요 성과**:
- TB급 대용량 데이터 분산 처리 및 분석
- 농산물 시장 트렌드 및 수급 예측 모델 개발
- 유산균 산업 현황 및 최적 발효 조건 분석
- 데이터 기반 정책 수립 지원
