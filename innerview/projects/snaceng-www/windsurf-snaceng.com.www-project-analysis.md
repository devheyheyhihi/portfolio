## 🚀 기술적 특징

### 1. 레이아웃 시스템
- PHP `require`를 통한 템플릿 시스템
- `RenderContents()` 함수로 콘텐츠 주입
- 공통 헤더/푸터 분리

### 2. 상태 관리
```php
global $us, $db, $is_paid, $is_old_paid;
```
- `$us`: 사용자 정보
- `$db`: 데이터베이스 객체
- `$is_paid`: 유료 사용자 여부
- `$is_old_paid`: 기존 유료 사용자 여부

### 3. 조건부 렌더링
```php
<?=($is_old_paid?"iblind":"")?>
```
- 유료 사용자에게는 무료 체험 버튼 숨김

### 4. OS 감지
```php
if(fn\g('os')=="ios") $us["os"]="ios";
if(fn\g('os')=="adr") $us["os"]="adr";
```
- 모바일 앱에서 접속 시 자동 리다이렉트
