# Cursor 프로젝트 분석 보고서

## 🔍 프로젝트 개요

**프로젝트명**: Annual (연차 관리 시스템)
**버전**: 0.0.0
**타입**: React 웹 애플리케이션
**빌드 도구**: Vite
**UI 프레임워크**: TailwindCSS + DaisyUI

## 📋 주요 기능

이 프로젝트는 기업의 **연차 및 경비 관리 시스템**으로, 다음과 같은 기능들을 제공합니다:

### 🏢 사용자 역할별 기능
- **일반 사용자(user)**: 연차 신청, 캘린더 조회, 예약 기능
- **팀장(leader)**: 연차 승인, 경비 신청, 직원 관리
- **관리자(admin)**: 전체 시스템 관리, 부서/팀/사용자 관리

### 📱 핵심 모듈
1. **연차 관리**
   - 연차 신청 (연차, 월차, 반차, 경조사, 병가, 기타)
   - 연차 승인 프로세스
   - 연차 현황 조회

2. **경비 관리**
   - 경비 신청 및 승인
   - 경비 내역 관리

3. **예약 시스템**
   - 회의실 예약
   - 요가 클래스 예약

4. **설정 관리**
   - 프로필 설정
   - 부서/팀 관리
   - 사용자 관리

## 🛠 기술 스택

### 프론트엔드
- **React 18.2.0**: 메인 프레임워크
- **Redux Toolkit**: 상태 관리
- **React Router Dom**: 라우팅
- **Vite**: 빌드 도구 및 개발 서버

### UI/UX
- **TailwindCSS**: 유틸리티 CSS 프레임워크
- **DaisyUI**: TailwindCSS 컴포넌트 라이브러리
- **Heroicons**: 아이콘 라이브러리
- **Chart.js**: 차트 라이브러리

### 추가 라이브러리
- **Axios**: HTTP 클라이언트
- **SWR**: 데이터 페칭
- **Moment.js/Day.js**: 날짜 처리
- **React Calendar**: 캘린더 컴포넌트
- **FilePond**: 파일 업로드
- **React Color**: 색상 선택기

```
│   └── RightSidebar.jsx  # 우측 사이드바
├── features/             # 기능별 모듈
│   ├── annual/           # 연차 관리
│   ├── calendar/         # 캘린더
│   ├── charts/           # 차트
│   ├── common/           # 공통 기능
│   ├── dashboard/        # 대시보드
│   ├── expenses/         # 경비 관리
│   ├── meetingRoom/      # 회의실 예약
│   ├── settings/         # 설정 관리
│   └── yoga/             # 요가 예약
├── pages/                # 페이지 컴포넌트
│   ├── protected/        # 인증이 필요한 페이지
│   └── 인증 관련 페이지들
└── routes/               # 라우팅 설정
```

## ⚙️ 주요 설정

### Vite 설정 (vite.config.js)
- React 플러그인 활성화
- 글로벌 상수 정의:
  - `MODAL_BODY_TYPES`: 모달 타입 정의
  - `ANNUAL_TYPE`: 연차 타입 (연차, 월차, 반차 등)
  - `ANNUAL_STATUS`: 연차 상태 (신청, 승인, 반려 등)
  - `RIGHT_DRAWER_TYPES`: 우측 드로어 타입

### TailwindCSS 설정
- DaisyUI 플러그인 사용
- 다크 모드 지원
- Typography 플러그인 활용

## 🚀 주요 기능 상세

### 연차 신청 시스템
- **다양한 연차 타입 지원**: 연차, 월차, 오전/오후 반차, 경조사, 병가, 기타
- **날짜 선택**: React Datepicker 활용
- **파일 첨부**: FilePond를 통한 드래그 앤 드롭 업로드
- **승인 프로세스**: 단계별 승인 시스템
- **팀 과반수 제한**: 팀원 50% 이상 연차 시 제한

### 캘린더 시스템
- **이벤트 표시**: 연차, 회의, 예약 등 통합 표시
- **색상 구분**: 이벤트 타입별 색상 코딩
- **상세 정보**: 우측 드로어를 통한 상세 정보 표시
